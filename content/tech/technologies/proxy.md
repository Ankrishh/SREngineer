## [HAProxy ](http://www.haproxy.org/)

- Load Balancer + Reverse Proxy - TCP & HTTP
- Dynamic routing
- Protocol agnostic - accepts anything sent over TCP

Metrics:
- Session Usage Frontend
- Session Usage Backend
- Errors - 4XX/5XX
- Queued requests
